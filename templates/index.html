Introduction
  • Grab a sudoku puzzle and showing it to the webcam until you see a green square on the screen.
  • Once a green square appeared, Then click "Display Captured Sudoku Puzzle" button below.
  • After click the button, the captured Sudoku puzzle is displayed in the 9 by 9 table below. Make sure each digit in your original Sudoku puzzle was recognized properly. If not, please fix them by filling in the corresponding cell.
  • Once you made sure every number is correct, then click "Solve" botton below.
  • Finally, the solution will be displayed in the "Result" section. If Qunantum Computer failed to solve the puzzle, then try another one.
Note
  • When video streaming does not start, plaese refresh the page.
  • Though we use the D-Wave quantum computers to solve Sudoku puzzles which has fascinating speed for the computation, many people access the computers so that we have to wait a queue for using them. So the total process will take time but usually it is done with in a minute.
  • Sometimes the quantum computers can not find the solution for the given Sudoku puzzle especially for difficult ones since quantum computers runs calculation for several times and pick up the best solution it found. In other words, it did not converged to an optima of the objective function.
